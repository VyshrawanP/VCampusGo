# Campus Helper (Unofficial)

A student-built web application to help VIT-AP students:
- Search faculty office locations
- View daily mess menu
- Get quick campus-related information

## Disclaimer
This is an **unofficial student project**.
- Not affiliated with VIT-AP University
- Data is based on publicly available information
- Information may change and is not guaranteed to be accurate

## Tech Stack
- HTML, CSS, JavaScript
- JSON-based data
- No backend / no database

## Why this project?
The official website is hard to navigate.
This project focuses on **speed, clarity, and student usability**.
